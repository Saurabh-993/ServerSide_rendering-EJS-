Common Input Attributes
Atrributes Working Examples
name Sabse zaroori. Isi naam se data server me jata hai. Name nahi to data nahi jayega name="email"
id CSS, JS aur <label for=""> se link karne ke liye id="email"
value By default kya likha ho value="default text"
placeholder Halka sa hint text, user likhega to hat jayega placeholder="Email daalo"
required Khali nahi chhod sakte. Submit nahi hoga <input required>
readonly User dekh sakta hai, edit nahi kar sakta value="Delhi" readonly
disabled Dikhega par grey ho jayega. Server me nahi jayega <input disabled>
Size Kitna lamba dikhe box, characters me size="30"
maxlength Kitne character daal sakte ho max "maxlength=10"
minlength Kam se kam kitne daalne padenge "minlength=5"
min/max Number, date, range ke liye min-max value min="18" max="99"
Step Number/range me kitne ka jump ho step="5" = 5,10,15...
pattern Regex se validation. Advance cheez pattern="[A-Za-z]{3}" = 3 letter hi
autocomplete on/off ya name, email = browser suggest kare autocomplete="email"
autofocus Page load hote hi ispe cursor chala jaye. Ek page me 1 hi <input autofocus>
