  2. Schema.org basically this tells google in details.

    So basically this is an JSON file that explains about the page , that is this page is an article , Product or Recipe. And through this google show rating and prices etc in search.


    Example of code is given below:-


                 
< script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "HTML Complete Guide",
  "author": {
    "@type": "Person",
    "name": "Dev Bhai"
  },
  "datePublished": "2026-04-07"
}
< /script>
