var( ):- it is used for having variables, also we can provide fallback values in it ,for eg. var(Htextcolor,red,blue) so here red and blue are fallbacks.

Dark Mode

for dark mode we have to use

        @media (prefer-color-scheme:dark){
            then you have to wrire all the variables that you have 
            writtern in normal document and provide them another value.
        }
        Note:-instead of dark you can also use light for light mode.

Note: If your device is in dark mode than your :root mode is dark (device understand default as light in such condition your default is dark so device understand dark as light and light as dark so you have to write prefer-color-scheme:light) than it will work.
